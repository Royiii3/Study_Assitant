# 大学生学习行为数据分析与学业预测系统

## 项目简介

本项目是一个基于Python的大学生学习行为数据分析与学业预测系统，包含数据预处理、统计分析、可视化展示和学业预测四大核心功能模块。专门针对金融专业学生学习行为数据设计。

## 功能特点

-  **数据加载**：支持上传CSV数据文件或自动生成模拟数据
- ⚙️ **数据预处理**：自动处理缺失值、异常值，标准化数据格式
-  **统计分析**：学习行为与成绩的相关性分析
-  **可视化展示**：多类型统计图表（柱状图、散点图、饼图、折线图）
-  **学业预测**：基于逻辑回归模型的学业等级预测
-  **模型评估**：展示预测模型的准确率和评估指标

## 环境要求

- Python 3.8 及以上版本
- 所需依赖库见 requirements.txt

## 安装与运行

### 1. 创建虚拟环境（推荐）

**Windows:**

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**macOS/Linux:**

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. 运行项目

```bash
streamlit run app.py
```

运行成功后，会自动打开浏览器，显示系统界面。

## 使用说明

### 数据加载

1. 点击左侧导航菜单的「数据加载」
2. 点击「上传CSV」按钮上传数据文件，或点击「生成模拟数据」按钮使用内置模拟数据
3. 查看数据预览和基本统计信息

### 数据预处理

1. 点击左侧导航菜单的「数据预处理」
2. 点击「执行预处理」按钮，系统会自动处理缺失值和异常值

### 统计分析

1. 点击左侧导航菜单的「统计分析」
2. 查看学习行为与成绩的相关性分析结果
3. 查看班级学习状态统计和学业等级分布

### 可视化展示

1. 点击左侧导航菜单的「可视化展示」
2. 查看各类统计图表：
   - 科目成绩分布柱状图
   - 学习行为与成绩关联散点图
   - 班级与学业等级分布饼图
   - 学期学习进度趋势折线图
   - 相关性热力图
   - 各班级学业等级分布堆叠柱状图

### 学业预测

1. 点击左侧导航菜单的「学业预测」
2. 先点击「训练预测模型」按钮训练模型
3. 输入个人学习行为数据（学习时长、作业完成度、考勤率、测验成绩、课堂参与度）
4. 点击「预测学业等级」按钮查看预测结果

### 模型评估

1. 点击左侧导航菜单的「模型评估」
2. 查看模型的训练集和测试集准确率
3. 查看分类报告和混淆矩阵

## 数据格式说明

CSV数据文件应包含以下字段：

| 字段 | 说明 |
|------|------|
| 学号 | 学生编号 |
| 班级 | 学生所在班级 |
| 性别 | 学生性别 |
| 学习时长 | 每周学习时长（小时） |
| 作业完成度 | 作业完成百分比 |
| 考勤率 | 考勤百分比 |
| 测验成绩 | 最近测验成绩 |
| 课堂参与度 | 1-5分 |
| PTA成绩 | 编程语言科目成绩 |
| 高数成绩 | 高等数学科目成绩 |
| 英语成绩 | 英语科目成绩 |
| Python成绩 | Python语言科目成绩 |
| 最终成绩 | 综合成绩 |
| 学业等级 | 优/良/中/差 |

## 技术栈

- **前端框架**：Streamlit
- **数据分析**：pandas, numpy
- **可视化**：matplotlib, seaborn
- **机器学习**：scikit-learn（逻辑回归）

## 跨平台兼容性说明

本项目支持在Windows、macOS和Linux操作系统上运行。关键注意事项：

1. **路径分隔符**：代码中已使用`os.path.join()`和正斜杠`/`处理路径，确保跨平台兼容
2. **中文字体**：可视化模块已配置SimHei和DejaVu Sans后备字体
3. **虚拟环境命令**：
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`
4. **Streamlit运行**：不同系统均使用`streamlit run app.py`命令
